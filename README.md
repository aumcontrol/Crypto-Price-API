# Crypto-Stock API

## Installation
1. Clone the repository.
2. Run `npm install` to install dependencies.
3. Create a `.env` file and provide the required configurations.

## Usage
Start the server using:
```
node server.js
```

API Endpoints:
- `GET /api/crypto/:ticker` - Fetch crypto price by ticker.
- `GET /api/stock/:ticker` - Fetch stock price by ticker (requires API setup).
